# 户口迁移,会改变籍贯

# 深A、沪A区别：

# 沪A股东卡，一个人只能有三个。 一个沪A股东卡，只能挂一个资金号。

深A股东卡，一个人能开无数个，但默认只开一个。 因为一个深A股东卡，可挂无数个资金账号。
